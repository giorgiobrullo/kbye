## About

kbye encodes any message into a sequence of dry, uninterested-sounding words. Send "ok k bye lol cool nice sure yeah" and your friend decodes it back to your actual message. Perfect for when you need to say something but want to look like you don't care.

Shareable URLs auto-update as you type. Enter to encode, Shift+Enter to decode.

## Quick Start

```bash
bun install
bun dev
bun run build
```
